Integration Project

This was the very first program that I created for my COP 1500 class, so it's definitely not the greatest, but it's what got me interested in coding.

This program is going to ask the user to choose between cookies, cake, or pies, and then has them choose one of 3 options provided for each. It is then going to ask them how many servings they would like, and then the program will do the calculations to provide the user with exact quantities.
